### [Yeonjin Kim] 토스트(Toast) 알림 시스템 구현 및 전역 상태 연결
사용자에게 작업의 성공, 실패, 경고 등의 상태를 시각적으로 즉각 전달하기 위해 토스트 알림 시스템이 도입되었습니다. **Yeonjin Kim** 님은 `Zustand`를 활용하여 전역 상태 관리를 구현하고, 다양한 상황에 대응할 수 있는 UI 컴포넌트와 유틸리티 함수를 제작했습니다.

**1. 전역 상태 관리 (`useToastStore`)**
토스트의 노출 여부와 메시지 내용을 관리하는 스토어를 구축하여 어디서든 쉽게 알림을 호출할 수 있도록 했습니다.

```typescript
// src/store/useToastStore.ts
export const useToastStore = create<ToastState>((set) => ({
  toast: null,
  showToast: (payload) => {
    const id = toastId++;
    set({ toast: { ...payload, id } });
  },
  hideToast: () => set({ toast: null }),
}));
```

**2. 에러 핸들링 유틸리티 통합**
Axios 에러 및 폼 검증 에러에서 메시지를 추출하여 토스트로 연결하는 유틸리티를 추가하여 코드의 재사용성을 높였습니다.

```typescript
// src/shared/utils/error.ts
export const getErrorMessage = (error: unknown, fallbackMessage = '잠시 후 다시 시도해주세요.') => {
  if (axios.isAxiosError<ErrorResponseData>(error)) {
    const responseMessage = error.response?.data?.message?.trim();
    if (responseMessage) return responseMessage;
    if (error.code === 'ERR_NETWORK') return '네트워크 연결을 확인한 뒤 다시 시도해주세요.';
  }
  return fallbackMessage;
};
```

**3. UI 컴포넌트 및 스타일링**
다양한 아이콘(Check, Exclamation, Info 등)과 테마를 적용하여 시인성을 확보했습니다. 특히 `ToastViewport`를 통해 포털 방식으로 최상단에 렌더링되도록 설계되었습니다.

```tsx
// src/shared/ui/common/Toast/ToastViewport.tsx
const ToastViewport = () => {
  const { toast, hideToast } = useToastStore(
    useShallow((state) => ({
      toast: state.toast,
      hideToast: state.hideToast,
    })),
  );

  if (!toast) return null;

  return (
    <Toast
      key={toast.id}
      title={toast.title}
      message={toast.message}
      // ... 생략
    />
  );
};
```

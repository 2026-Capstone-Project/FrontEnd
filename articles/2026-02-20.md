**주요 변경 사항:**
- **API 레이어 구축**: `fetchTodayBriefing`, `fetchReminders` 함수를 추가하여 홈 화면에 필요한 데이터를 호출합니다.
- **타입 시스템 강화**: `BriefingResponse`, `ReminderResponse` 등 서버 데이터 구조에 맞춘 인터페이스를 정의했습니다.

```typescript
// src/shared/api/home/home.ts
export const fetchTodayBriefing = async (): Promise<BriefingResponse> => {
  const res = await axiosInstance.get<BriefingResponse>('/api/v1/briefings')
  return res.data
}

export const fetchReminders = async (): Promise<ReminderResponse> => {
  const res = await axiosInstance.get<ReminderResponse>('/api/v1/reminders')
  return res.data
}
```

### [Yeonjin Kim] API 경로 최적화 및 레이아웃 시스템 리팩터링
코드 리뷰 피드백을 반영하여 전반적인 프로젝트 구조를 개선했습니다. 기존의 불필요한 `/api/v1` 접두사를 제거하여 엔드포인트를 간결하게 정리했으며, 설정(Settings) 페이지 전용 레이아웃을 분리하여 라우팅 구조를 체계화했습니다.

- **API 엔드포인트 일괄 수정**: Axios 인터셉터 및 개별 API 호출부에서 `/api/v1` 경로를 삭제하여 백엔드 프록시 및 API 게이트웨이 설정에 맞게 최적화했습니다.

### [yujin5959] feat: 선제적 제안 관련 api 연결

**yujin5959** 개발자는 선제적 제안(Proactive Suggestion) 기능을 구현하기 위해 백엔드 API와의 인터페이스를 구축하고, 이를 홈 화면에 통합하는 작업을 수행했습니다.

#### 1. API 통신 레이어 및 타입 정의
기존의 브리핑 및 리마인더 API 외에 선제적 제안을 위한 `suggestionApi`를 추가했습니다. 특히 응답 데이터의 구조를 명확히 하기 위해 `Suggestion` 인터페이스와 `SuggestionListResponse` 타입을 정의하여 타입 안정성을 확보했습니다.

```typescript
// src/shared/types/home/home.ts
export interface Suggestion {
  id: number
  content: string
}

export interface SuggestionListResponse {
  isSuccess: boolean
  code: string
  message: string
  result: {
    details: Suggestion[]
  }
}
```

```typescript
// src/shared/api/home/home.ts
export const suggestionApi = {
  getSuggestions: async (): Promise<SuggestionListResponse> => {
    const res = await axiosInstance.get<SuggestionListResponse>('/suggestions')
    return res.data
  }
}
```

**시니어 개발자 총평:**
이번 커밋을 통해 선제적 제안 기능의 뼈대가 완성되었습니다. 특히 `TCommonResponse`를 활용한 공통 응답 처리와 명확한 타입 정의는 향후 유지보수 측면에서 긍정적인 역할을 할 것입니다. 다만, `HomePage.styles.ts`에 추가된 개발용 코드는 기능 검증 완료 후 반드시 제거될 수 있도록 코드 리뷰 단계에서 다시 한번 체크가 필요합니다. 내일부터는 연동된 API를 바탕으로 실제 사용자 데이터가 제안 UI에 어떻게 렌더링되는지 성능 최적화 관점에서 살펴볼 예정입니다.

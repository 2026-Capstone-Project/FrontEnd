## 🌿 Branch: develop

### [Yeonjin Kim] 레이아웃 재설정 및 API 링크 수정 (PR #36)

Yeonjin Kim 님은 전역적인 API 호출 경로 수정과 라우팅 구조 최적화 작업을 수행했습니다. 

1. **API 엔드포인트 정규화**: 
   기존의 모든 API 호출에서 `/api/v1` 접두사를 제거했습니다. 이는 백엔드 프록시 설정 변경 또는 API 게이트웨이 구조 변화에 대응하기 위한 조치로 보입니다. `axiosInstance` 호출부뿐만 아니라 CSRF 토큰 획득 및 소셜 로그인 콜백 URL까지 모두 수정되었습니다.

   ```typescript
   // src/app/main.tsx
   - const reissueRes = await axiosInstance.post('/api/v1/security/reissue-cookie', {})
   + const reissueRes = await axiosInstance.post('/security/reissue-cookie', {})

   // src/shared/api/settings/settings.ts
   - const API_BASE = '/api/v1/settings'
   + const API_BASE = '/settings'
   ```

2. **라우팅 및 레이아웃 구조 개선**:
   `MainRoutes`에 포함되어 있던 `SettingsPage`를 별도의 `SettingRoutes`로 분리하고, `SettingLayout`을 신규 도입했습니다. 이를 통해 설정 페이지만의 독립적인 UI 흐름을 제어할 수 있게 되었습니다.

   ```typescript
   // src/routes/SettingRoutes.tsx
   const SettingRoutes: RouteObject = {
     path: '/settings',
     element: <SettingLayout />,
     children: [
       {
         index: true,
         element: <SettingsPage />,
       },
     ],
   }
   ```

3. **UI 컴포넌트 위치 최적화**:
   `AIChatModal`의 위치 방식을 `fixed`에서 `absolute`로 변경하고, `HomePage`의 반응형 레이아웃을 `media` 유틸리티를 사용하는 방식으로 개선하여 코드의 일관성을 높였습니다.

### [yujin5959] 홈 페이지 API 연결 및 방어적 프로그래밍 적용

yujin5959 님은 홈 페이지에서 필요한 실시간 데이터 연동 작업을 마무리했습니다.

1. **홈 페이지 API 연동**:
   `fetchTodayBriefing`과 `fetchReminders` API를 연동하여 사용자의 오늘 일정 요약 및 리마인더 정보를 실시간으로 받아오도록 구현했습니다. `useCustomQuery`를 활용해 데이터 페칭 로직을 추상화했습니다.

2. **방어적 프로그래밍 및 코드 최적화**:
   코파일럿 코드 리뷰를 바탕으로 리마인더 데이터가 없을 경우를 대비해 기본값(`[]`)을 설정하고, 불필요하게 중복 정의된 `BriefTime` 타입을 기존 `BriefingTime` 타입으로 통합하여 타입 안정성을 확보했습니다.

   ```typescript
   // src/pages/main/HomePage/HomePage.tsx
   const { data: reminders = [] } = useCustomQuery(['reminders'], fetchReminders, {
     select: (response) => response.result,
     refetchInterval: 60000, // 1분 간격 갱신
   })

   // src/shared/types/home/home.ts
   - export interface BriefTime { ... }
   + export type BriefTime = BriefingTime
   ```

3. **신규 API 및 타입 정의**:
   홈 브리핑 기능을 위한 `BriefingResponse`, `ReminderResponse` 등의 인터페이스를 정의하고, `src/shared/api/home/home.ts`를 신설하여 관련 API 로직을 응집시켰습니다.

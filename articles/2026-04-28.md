## 🌿 Branch: feature/#95

### 1. 반복 설정 및 공통 UI 스타일 정밀 조정 (작성자: Yeonjin Kim)
반복 설정 입력 창의 시각적 불균형을 해소하고, 체크박스의 상태에 따른 피드백을 강화했습니다. 특히 입력 필드의 너비를 최적화하여 모바일 및 데스크톱 환경에서의 가독성을 개선했습니다.

*   **Checkbox**: 체크 여부에 따라 보더와 배경색이 유연하게 변경되도록 로직을 수정했습니다.
*   **Input Width**: 반복 주기 및 종료 횟수 입력 필드의 너비를 조정하여 불필요한 여백을 제거했습니다.

```typescript
// src/shared/ui/common/Checkbox/Checkbox.style.tsx
export const Checkbox = styled.input<{ isChecked?: boolean }>`
  appearance: none;
  width: 100%;
  height: 100%;
-  border: none;
+  border: ${(props) => (props.isChecked ? 'none' : '1px solid #c4c4c4')};
  border-radius: 4px;
-  background-color: ${(props) => (props.isChecked ? theme.colors.primary2 : '#D2D3D2')};
+  background-color: ${(props) => (props.isChecked ? theme.colors.primary2 : 'transparent')};
  cursor: pointer;
  transition: background-color 150ms ease;
`
```

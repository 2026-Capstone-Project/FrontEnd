**1. Axios 인터셉터를 통한 토큰 재발급 로직 구현**
기존에 미비했던 401(Unauthorized) 에러 발생 시의 자동 토큰 재발급 로직을 `axios.ts`에 추가했습니다. 이를 통해 사용자가 세션 만료 시 다시 로그인해야 하는 번거로움을 줄이고 서비스 연속성을 보장합니다.

```typescript
// src/shared/api/axios.ts 변경 사항
axiosInstance.interceptors.response.use(
  (response) => response,
  async (error) => {
    const originalRequest = error.config

    if (error.response?.status === 401 && !originalRequest._retry) {
      originalRequest._retry = true

      try {
        await axios.post(
          `${import.meta.env.VITE_API_SERVER_URL}/security/reissue-cookie`,
          {},
          { withCredentials: true }
        )
        // 재발급 성공 시 기존 요청 재시도 로직 추가 예정
      } catch (reissueError) {
        return Promise.reject(reissueError)
      }
    }
    return Promise.reject(error)
  }
)
```

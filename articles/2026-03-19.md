#### 1. 캘린더 툴바 UX 및 접근성 개선
기존에 단순 텍스트로 존재하던 날짜 레이블을 버튼 요소로 변경하여 클릭 시 '오늘(TODAY)'로 즉시 이동할 수 있는 기능을 추가했습니다. 또한, 키보드 사용자를 위한 `focus-visible` 스타일을 정의하여 웹 접근성을 준수했습니다.

```typescript
// src/features/Calendar/components/CalendarToolbar/CalendarToolbar.tsx
<button
  type="button"
  className="date-label"
  onClick={() => onNavigate('TODAY')}
  title="오늘로 이동"
  aria-label="오늘로 이동"
>
  {formattedLabel}
</button>

// src/features/Calendar/components/CalendarToolbar/CalendarToolbar.style.ts
.date-label {
  border: none;
  background: none;
  padding: 0;
  cursor: pointer;

  &:focus-visible {
    outline: 2px solid ${theme.colors.primary2};
    outline-offset: 6px;
    border-radius: 10px;
  }
}
```

#### 4. UI 일관성 및 폼 데이터 전달 보완
일정 추가 폼에서 수정 모드(`isEditing`) 상태가 누락되었던 점을 보완하였고, 삭제 확인 및 변경 사항 저장 확인 모달의 그림자(box-shadow) 값을 조정하여 시각적인 깊이감을 통일했습니다.

### 2. [Yeonjin Kim] 제목 추천어 API 연결 및 UX 최적화
사용자가 일정이나 할 일의 제목을 입력할 때, 과거 기록을 바탕으로 추천어를 제공하는 기능을 구현했습니다. 이를 위해 백엔드 API를 연결하고, 클라이언트 측에서 불필요한 API 호출을 방지하기 위한 최적화 기법을 적용했습니다.

**주요 기술적 분석:**
*   **Throttling 적용:** `useThrottledValue` 커스텀 훅을 구현하여, 사용자의 입력이 잦을 때 API 호출 횟수를 제한함으로써 서버 부하를 줄이고 성능을 개선했습니다.
*   **React Query 연동:** `calendarKeys`와 `todoKeys`에 `titleHistory`를 추가하여 추천어 데이터를 효율적으로 관리하고 캐싱합니다.
*   **UI/UX 디테일 수정:** `SelectColor` 드롭다운의 위치를 `absolute`로 조정하여 레이아웃 깨짐 현상을 방지하고, 중요도(Priority) 옵션을 상수로 관리하여 코드의 응집도를 높였습니다.

```typescript
// src/shared/hooks/common/useThrottledValue.ts
import { useEffect, useRef, useState } from 'react'

export const useThrottledValue = <T>(value: T, delay = 150) => {
  const [throttledValue, setThrottledValue] = useState(value)
  const lastUpdatedAtRef = useRef(0)

  useEffect(() => {
    const now = Date.now()
    const remainingTime = delay - (now - lastUpdatedAtRef.current)
    
    if (remainingTime <= 0) {
      setThrottledValue(value)
      lastUpdatedAtRef.current = now
    } else {
      const timeout = setTimeout(() => {
        setThrottledValue(value)
        lastUpdatedAtRef.current = Date.now()
      }, remainingTime)
      return () => clearTimeout(timeout)
    }
  }, [value, delay])

  return throttledValue
}
```

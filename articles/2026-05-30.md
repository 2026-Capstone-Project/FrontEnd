  * **UI/UX 및 스타일링 (`Friend.styles.ts`, `FriendItem.tsx`, `FriendListSection.tsx` 등)**: Emotion을 활용하여 반응형 그리드 레이아웃(`PageLayout`, `Column`)을 구현했습니다. 태블릿 및 모바일 환경(1024px, 800px 미디어 쿼리)에 대응할 수 있도록 유연한 Flex 구조를 채택했습니다.

  * **상태 관리 (`useFriendsMutations.ts`)**: TanStack Query(React Query)의 `useMutation`을 활용하여 비동기 요청 성공 시 `queryClient.invalidateQueries`를 호출함으로써, 클라이언트의 데이터 정합성을 실시간으로 유지하도록 설계했습니다.
  * **컴포넌트 분리**: 일정 정보를 보여주는 `ScheduleItem`과 `SharedScheduleItem`을 분리하여 데이터의 성격에 맞는 렌더링을 보장합니다.

```typescript
// src/shared/hooks/friends/useFriendsMutations.ts 일부
import { useMutation, useQueryClient } from '@tanstack/react-query'
import { friendApi, friendRequestApi } from '@/shared/api/friends/friends'

export function useFriendMutations() {
  const queryClient = useQueryClient()

  const acceptMutation = useMutation({
    mutationFn: (friendRequestId: number) => friendRequestApi.acceptRequest({ friendRequestId }),
    onSuccess: () => {
      queryClient.invalidateQueries(['friends'])
      queryClient.invalidateQueries(['friendRequests'])
    }
  })
  // ... 수락, 거절, 삭제 Mutation 정의
}
```

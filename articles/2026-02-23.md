### [Yeonjin Kim] 변경된 API 스펙 반영 및 데이터 포맷팅 최적화
백엔드 API의 변경 사항에 맞춰 일정 수정(PATCH) 시 `occurrenceDate`와 `scope`를 쿼리 파라미터로 분리하여 전송하도록 구조를 변경했습니다. 또한, 시간 데이터 전송 시 초 단위까지 포함(`YYYY-MM-DDTHH:mm:ss`)하도록 포맷을 통일하여 데이터 정밀도를 높였습니다.

```typescript
// src/shared/api/calendar/api.ts
export const patchEvent = async (
  eventId: number,
  eventData: { /* ... */ },
  params: {
    occurrenceDate: string
    scope?: Extract<RecurrenceEventScope, 'THIS_EVENT' | 'THIS_AND_FOLLOWING_EVENTS'>
  },
) => {
  // params를 통해 occurrenceDate와 scope를 별도로 전달하도록 변경
  const { data } = await axiosInstance.patch(`/events/${eventId}`, eventData, { params })
  return data
}
```

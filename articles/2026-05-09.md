#### 1. 공유 일정 식별을 위한 데이터 구조 및 UI 반영
일정 데이터 타입에 `isShared` 속성을 추가하고, 캘린더 뷰에서 공유된 일정임을 한눈에 알 수 있도록 `people.svg` 아이콘을 적용했습니다. 또한, 텍스트가 잘리지 않도록 스타일을 조정했습니다.

```typescript
// src/shared/types/calendar/types.ts
export type Event = {
  // ... 기존 속성
  isAllDay: boolean
  color: EventColorType
  recurrenceGroup: RecurrenceGroup | null
  isShared?: boolean // 공유 상태 추가
}

// src/features/Calendar/components/CustomEvent/CustomEvent.style.ts
export const EventTitle = styled.div`
  font-weight: 400;
  font-size: 12px;
- width: 38px;
+ width: 75px; // 아이콘 추가 및 가독성을 위한 너비 확장
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
```

#### 2. 공유 일정 관리 로직 구현 (`useCalendarEvents`)
이벤트의 공유 상태를 업데이트할 수 있는 `updateEventShared` 함수를 훅에 추가하여, 에디터에서 변경된 공유 설정이 캘린더 상태에 즉각 반영되도록 했습니다.

```typescript
// src/features/Calendar/hooks/useCalendarEvents.ts
const updateEventShared = useCallback((eventId: CalendarEvent['id'], isShared: boolean) => {
  setEvents((prev) =>
    prev.map((event) => (event.id === eventId ? { ...event, isShared } : event)),
  )
}, [])
```

#### 3. 일정 공유 패널 및 친구 검색 UI 신설
일정 에디터 내에서 친구를 검색하고 공유 대상으로 지정할 수 있는 `ShareSchedulePanel`과 `SearchFriend` 컴포넌트를 구현했습니다. 이를 위해 `share` 전용 컬러 테마를 정의하고 모의 데이터(Mock Data)를 활용한 검색 필터링 로직을 구축했습니다.

```typescript
// src/shared/styles/theme.ts
export const colors = {
  // ...
  share: {
    base: '#ECEBFF',
    point: '#594FCA',
  },
};

// src/shared/utils/scheduleShareFriend.ts
export const filterScheduleShareFriends = (
  friends: ScheduleShareFriend[],
  keyword: string,
): ScheduleShareFriend[] => {
  const trimmedKeyword = keyword.trim().toLowerCase()
  if (!trimmedKeyword) return []
  return friends.filter(
    (friend) =>
      friend.userName.toLowerCase().includes(trimmedKeyword) ||
      friend.email.toLowerCase().includes(trimmedKeyword),
  )
}
```

#### 4. 에디터 컴포넌트 및 훅 리팩토링
기존에 `common` 폴더에 혼재되어 있던 UI 컴포넌트들을 `scheduleTodo`라는 명확한 도메인 폴더로 이동시키고, 에디터의 복잡한 상태 관리를 위해 `useEditorDraft`, `useEditorRegistry`, `useEditorTypeSync` 등의 커스텀 훅을 분리하여 유지보수성을 향상시켰습니다.

*   **주요 리팩토링 사항:**
    *   `SelectColor`, `SearchPlace`, `TitleSuggestionInput` 등의 경로 최적화.
    *   `EditorModalLayout`의 모바일 패딩 및 레이아웃 버그 수정.
    *   `CustomDatePicker` 내비게이션 버튼 스타일 개선.
    *   `Checkbox`의 선택 전/후 시각적 피드백 강화.

```typescript
// src/shared/ui/common/Checkbox/Checkbox.style.tsx
export const Checkbox = styled.input<{ isChecked?: boolean }>`
  appearance: none;

  height: 100%;

- background-color: ${(props) => (props.isChecked ? theme.colors.primary2 : '#D2D3D2')};

```
